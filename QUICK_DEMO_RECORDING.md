# 🎬 クイックスタート: デモ録画

## 📦 インストール (初回のみ)

```bash
# 依存パッケージをインストール
pip install playwright Pillow

# Chromiumブラウザをインストール
playwright install chromium

# ffmpeg (高品質GIF用 - オプション)
brew install ffmpeg
```

## 🚀 録画実行

```bash
# 1コマンドで完了！
python record_demo.py
```

## ✅ 出力

- `lumen_demo.webm` - Web用 (高品質・小サイズ)
- `lumen_demo.gif` - 汎用 (互換性最高)

## 🎯 デモ内容 (30秒以内)

1. ファイルアップロード
2. リスク分析
3. 結果確認
4. 詳細表示

## 💡 特徴

- ✨ 人間らしい滑らかなマウス移動 (Bezier曲線)
- 📐 HD解像度 (1280x720)
- 🤖 完全自動化
- 🎨 プロフェッショナルな仕上がり
